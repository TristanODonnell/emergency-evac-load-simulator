# Emergency Evacuation Load Simulator

This project explores constrained evacuation scenarios using a deterministic, graph-based simulation model.

The simulator evaluates how different evacuation strategies perform under limited transportation capacity, time pressure, and uneven population distribution.

## Status
Planning & system design phase.

## Scope
- Deterministic evacuation simulation
- Synthetic data only
- Config-driven scenarios and strategies
- No real GIS, traffic, or population data

## Non-Goals
- Real-time systems
- Machine learning
- UI or visualization-heavy components

## Next Steps
- Formalize system model
- Define input configuration schema
- Design baseline evacuation scenarios
